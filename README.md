Este projeto foi criado em 2025, durante o terceiro ano do Ensino Médio, para a Mostra Científica Cultural da Escola Estadual de Ensino Médio Caic Madezatti, em parceria com meu colega e amigo Lucas Prestes Vargas.

O projeto consiste na criação de uma rede Wi-Fi falsa que redireciona o usuário para uma réplica da página de login do EscolaRS, simulando o funcionamento de um stealer de dados. O objetivo foi demonstrar, na prática, como ataques de engenharia social e redes maliciosas podem comprometer a segurança digital dos usuários.

Vale ressaltar que, devido às políticas da escola e a princípios éticos, nenhum dado sensível foi coletado, armazenado ou utilizado durante o projeto. Toda a demonstração foi realizada apenas para fins educativos e de conscientização sobre segurança digital.

Funcionamento (passo a passo)

O usuário conecta-se à rede Wi-Fi aberta criada pelo projeto.

Ao acessar a rede, ele é direcionado para uma página solicitando login no EscolaRS.

Ao clicar no botão de login, o usuário é redirecionado para uma página de aviso informando que se trata de um projeto escolar, garantindo que nenhuma informação será utilizada ou colocará o usuário em risco.

A ação de clicar no botão era registrada apenas para contabilizar quantos usuários interagiram com a demonstração.

Os administradores do projeto possuíam uma área de controle, onde era possível visualizar estatísticas básicas, como o número de interações realizadas pelos usuários.

O projeto teve como objetivo demonstrar vulnerabilidades comuns em redes públicas e conscientizar sobre a importância da segurança digital.

Graças ao trabalho desenvolvido, o projeto recebeu nota 10 na feira científica.
